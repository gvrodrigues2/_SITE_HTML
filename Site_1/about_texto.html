Tags de Texto

No HTML, as tags h1 até h6 são utilizadas para criar cabeçalhos de diferentes níveis. Esses cabeçalhos variam em tamanho e importância, sendo h1 o mais importante e h6 o menos importante.

Aqui está uma breve definição de cada uma delas:
  • h1: É o mais importante e geralmente usado para títulos principais na página.
  • h2: É um pouco menos importante do que h1 e é usado para subtítulos importantes.
  • h3: Menos importante do que h2, usado para subtítulos de h2.
  • h4: Menos importante do que h3, usado para subtítulos de h3.
  • h5: Menos importante do que h4, usado para subtítulos de h4.
  • h6: É o menos importante e geralmente usado para subtítulos de h5.
  • p: É usada para definir um parágrafo de texto.
  • mark: É usada para destacar parte do texto dentro de um parágrafo ou de qualquer outra tag de texto.
  • small: É usada para diminuir o tamanho do texto dentro de um parágrafo ou de qualquer outra tag de texto.
  • i: É usada para deixar partes do texto em itálico.
  • u: É usado para deixar partes do texto sublinhado.
  • strong: É usado para deixar partes do texto em negrito.
  • sub: É usada para criar uma linha horizontal ou uma regra de separação em uma página da web.
  • sup: É usado para criar texto subscrito.
  • blockquote: É usado para criar texto subreescrito.
